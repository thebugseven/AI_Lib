# AI_Lib – Implémentation d’une Intelligence Artificielle

## Description

**AI_Lib** est une librairie que je développe pour recréer et comprendre les mécanismes fondamentaux de l’intelligence artificielle.  
L’objectif principal est pédagogique : repartir de zéro et implémenter les concepts clés sans dépendances lourdes.

Cette librairie peut inclure (ou inclura) :
- Réseaux de neurones
- Algorithmes d’apprentissage supervisé
- Fonctions d’activation
- Optimisation (gradient descent, etc.)
- Gestion de datasets

---

## Objectifs

- Comprendre en profondeur le fonctionnement des IA
- Reproduire les bases des frameworks comme TensorFlow / PyTorch
- Avoir une implémentation simple, lisible et modifiable
- Servir de base pour des projets plus complexes
